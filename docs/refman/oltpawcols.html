NAME

oltpawcols − Create or recreate the aw_cols table

SYNOPSIS

oltpawcols

USAGE

The oltpawcols command takes no options and will (drop and) create the aw_cols table and its indexes. This is primarily used if you need to change its size by setting a new value for the aw_cols_256MB_doublings parameter.
