NAME

rwlman − RWP*Load Simulator overview of rwl man pages

SYNOPSIS

rwlman

rwlman page

DESCRIPTION

The rwlman command without any options shows this overview of the man pages for the RWP*Load Simulator, rwloadsim. As a whole, these man pages constitute the reference manual for rwloadsim. This reference manual is split into several major parts with the following top pages:

rwloadsim

The reference manual for the actual rwloadsim executable.

rwlprogram

The reference manual for the rwl programming language.

directive

The reference manual for the directives used to modify the behavior of rwloadsim.

statistics

This page describes how rwloadsim saves execution statistics.

utilities

This page describes various utilities included with rwloadsim as rwl scripts.

oltp

This page gives an introduction and overview of the oltp workload.

The rwlman script simply calls the normal man(3) utility with a proper MANPATH setting. Although possible, you are recommended not copying the man pages to your normal man page location such as /usr/share/man

BUGS

There is no consistent way to get brief command help. For rwloadsim itself, -h or --help is the option to output help while -H is used to output help from useroptions. Almost all oltp commands accept -H to get help, many also accept --help, some also -h for that purpose while -h is used to provide actual argument values for a few commands. The best way to get command syntax in a consistent way is to use rwlman; the second best is to try with a --help option, which works in all frequently used commands. This is confusing.

Some oltp commands accept both single letter and gnu style long options, which is also confusing.

AUTHOR

The primary developer and maintainer of the RWP*Load Simulator and its man pages is Bjørn Kisbye Engsig. The preferred way to contact him about rwloadsim is via the issues page on github.
